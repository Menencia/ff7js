# FF7js

**Status**: Work In Progress

## Testing

Please follow the steps after cloning :

- Install node libs: `npm install`

- Install JS libs: `bower update`

- Install Gulp for compiling: `npm install gulp -g`

- Compile ES6 JS files to ES5: `gulp`
